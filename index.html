This example should change the color of the shape on the click event.

Here are the properties of Shape:

  • id : unique id created for each element (for testing purposes)
  • vx : the velocity value of X
  • vy : the velocity value of Y
  • x : initial x position
  • y : initial y position
  • width : radius * 2
  • height : radius * 2
  • radius : radius of circle
  • fill : fill color
  • stroke : stroke color
  • strokewidth : stoke value

There are random values for all of these.
